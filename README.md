# Excel Project: Healthcare Performance Analysis

## Project Overview
This project analyzes healthcare data using **Microsoft Excel** to uncover insights related to patient demographics, medical conditions, lifestyle factors, and healthcare costs. The project demonstrates strong skills in **data cleaning, Excel functions, Pivot Tables, and interactive dashboard creation**.

---

## Objectives
- Analyze healthcare costs across different disease categories
- Study the relationship between BMI, diabetes levels, and medical charges
- Compare cancer history among smokers and non-smokers
- Build an interactive Excel dashboard for data-driven insights

---

## Tools & Technologies
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Excel Dashboards and Slicers

---

## Excel Functions Used
- INDEX  
- MATCH  
- IF  
- MODE  
- VLOOKUP  
- HLOOKUP  

---

## Dataset Description
The project integrates multiple healthcare datasets, including:
- Customer demographic details
- Medical examination data (BMI, HBA1C, diabetes level)
- Hospitalization details
- Lifestyle factors (smoking status)
- Surgery history and medical charges

---

## Data Cleaning & Preparation
- Cleaned missing and inconsistent data values
- Used lookup functions to merge multiple datasets
- Created derived fields such as BMI category, age, and surgery count
- Standardized categorical data for accurate analysis

---

## Analysis Performed
- Weight status analysis (Normal, Overweight, Obesity, Underweight)
- Diabetes level analysis (Normal, Prediabetes, Diabetes)
- Cancer history comparison between smokers and non-smokers
- Surgery count and hospitalization tier analysis
- Total medical charges by disease category

---

## Dashboard Features
- Interactive slicers for:
  - Diabetes Level
  - Smoking Status
- Visualizations include:
  - Total Charges by Diabetes Level
  - Customer Distribution by Weight Status
  - Cancer History Distribution
- Dynamic KPIs for quick decision-making

---

## Key Insights
- Obesity and diabetes patients incur higher medical charges
- Smokers show a higher cancer prevalence compared to non-smokers
- Prediabetes patients contribute lower total medical charges
- Weight status strongly correlates with diabetes levels
